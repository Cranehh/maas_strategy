# 方案二：城市尺度MaaS应用潜力估计与空间分析

## 标题

**中文:** 北京市MaaS应用潜力估计：融合选择模型、合成人口与基于活动出行需求的空间微观仿真

**English:** Estimating the Adoption Potential of MaaS in Beijing: A Spatially Explicit Microsimulation Integrating Choice Models, Synthetic Population, and Activity-Based Travel Demand

---

## 核心研究问题

北京MaaS的理论应用潜力是多少？有多少居民会在各订阅层级采纳MaaS？他们在哪里？具有什么出行特征？不同政策情景下潜力如何变化？

---

## 方法论详细设计

### Step 1: 微观仿真数据准备 (Ch4 + Ch5)

- **合成人口** → 每个个体的社会经济属性
- **活动计划** → 每个个体的月度出行指标衍生：
  - 月方式频率向量 $(N_{metro}, N_{bus}, N_{ride}, N_{bike}, N_{car})$
  - 月出行总支出（基于距离 × 单价）
  - 通勤距离、出行链复杂度
  - 可达性指标（居住地到最近地铁站距离等，利用交通网络数据）

### Step 2: 两阶段采纳漏斗 (Ch1 + Ch2)

这是本方案的核心创新——区分两个决策阶段：

#### 阶段1 — 初次采纳决策（Ch1参数）

对每个个体 $i$ 的代表性出行，用 Ch1 的 DCM 计算：

$$p_{try}(i) = P(\text{选择MaaS方式} \mid \text{个体属性}, \text{出行属性})$$

月度尝试概率：

$$P_{try,month}(i) = 1 - (1 - p_{try}(i))^{N_{trips,month}(i)}$$

#### 阶段2 — 订阅转移决策（Ch2参数）

对已"尝试"MaaS的个体，用 Ch2 的潜在类别模型计算：

$$p_{subscribe}(i \mid class\ c) = P(\text{订阅套餐} \mid \text{已尝试}, \text{所属意愿类别})$$

考虑类别异质性：不同潜在类别的转移概率差异。

#### 综合采纳概率

$$P_{total}(i) = P_{try,month}(i) \times p_{subscribe}(i)$$

### Step 3: 套餐选择与附加值 (Ch3)

定义候选套餐菜单（基于 Ch3 研究中已测试的套餐类型）：

| 套餐类型 | 内容 |
|---------|------|
| **地铁导向套餐** | 地铁不限次 + 公交 N 次 + 共享单车 N 次 |
| **公交导向套餐** | 公交不限次 + 地铁 N 次 |
| **多模式套餐** | 各方式均衡配额 |
| **灵活套餐** | 积分制，可自由分配 |

- 对每个个体计算各套餐的 $AV(i, b)$
- 个体选择 $\max_b AV(i, b)$，若最大值 $> 0$ 则订阅该套餐

### Step 4: 空间聚合与政策分析 (新贡献)

#### 产出指标（TAZ/区级别）

| 指标 | 计算方法 |
|------|---------|
| 采纳率 | $\sum P_{total}(i) / N_{TAZ}$ |
| 订阅者数量 | $\sum \mathbb{I}[P_{total}(i) > threshold]$ |
| 月收入估计 | $\sum P_k \times \text{订阅概率}_k$ |
| 方式转移量 | MaaS采纳者的方式替代：$\Delta N_{car}, \Delta N_{metro}, \ldots$ |
| 碳减排量 | $\sum (\Delta N_{car} \times d_{car} \times EF_{car}) - (\Delta N_{PT} \times d_{PT} \times EF_{PT})$ |
| 公平性指标 | 各收入组、各区的采纳率差异 (Gini) |

#### 政策情景（5个）

| 情景 | 描述 |
|------|------|
| **A: 基准** | 现行票价、无补贴 |
| **B: 碳激励加倍** | 碳积分价值 ×2 |
| **C: 五环拥堵收费** | 私家车成本 +15–25元/天 |
| **D: 服务质量提升** | 公交/地铁接驳等待时间 −30% |
| **E: 低收入定向补贴** | 月收入 <5000元 的居民套餐半价 |

### Step 5: 目标差距分析

- **对标北京政府目标:** 600万日活用户（2025年目标）
- **计算:** 在什么政策组合下可达到此目标？
- **识别"最后一英里"群体:** 那些接近但未达到采纳阈值的人群特征

## 创新点

1. **两阶段采纳漏斗** — 文献首次在城市尺度区分"试用"和"订阅"两个决策阶段
2. **政策–采纳–碳减排全链条量化** — 从政策输入直接推到碳减排产出
3. **目标差距分析** — 直接服务于北京MaaS政策制定
4. **空间不平等揭示** — 中心城区 vs 郊区的MaaS获益差异
